# TypeScript Code Reading Assistant

Đây là công cụ hỗ trợ đọc hiểu code TypeScript.

Khi tham gia vào dự án TypeScript có sẵn, công cụ này giúp giải quyết các vấn đề sau khi đọc code hiện có:

- Không hiểu rõ cú pháp TypeScript
  - "`??` là gì?"
  - "`#name` với dấu `#` là comment à?"
  - "`<></>` là gì?"
- Không biết tên tính năng ngôn ngữ, không biết cách tìm kiếm
  - "Dấu `|` trong `string | undefined` gọi là gì?"
  - "`value as any` là tính năng gì?"
  - "`[K: string]: string` là gì?"

## Cách sử dụng

1. Copy paste code TypeScript muốn đọc hiểu vào editor bên trái
2. Click vào phần code muốn tìm hiểu
3. Ý nghĩa của code sẽ hiển thị ở vùng bên phải

## Thông tin bảo mật

- Tất cả code được xử lý offline, không gửi lên server. Bạn có thể yên tâm.
- Code được encode Base64 và trở thành URL hash. Khi chia sẻ URL, hãy chú ý đến thông tin bảo mật.
