# Enum chuỗi (string enum)

Trong kiểu liệt kê của TypeScript, giá trị của member cũng có thể là chuỗi. Enum được cấu thành từ chuỗi được gọi là enum chuỗi (string enum).

```ts twoslash
enum Direction {
  Up = "UP",
  Down = "DOWN",
  Left = "LEFT",
  Right = "RIGHT",
}
```
