# Array literal

Trong JavaScript có thể viết array bằng array literal. Array literal sử dụng dấu ngoặc vuông (`[`, `]`).

```js twoslash
[1, 2, 3];
```

Array literal cũng có thể viết xuống dòng ở điểm phân cách các phần tử. Phần tử cuối cùng có thể có dấu phẩy.

<!--prettier-ignore-->
```js twoslash
[
  1,
  2,
  3,
]
```
