* unknownはTypeScriptの型
* unknownはどんな型も代入できる型
* unknown型はanyを除いて他の型には代入できない
* 同様にunknown型は、プロパティアクセスや関数呼び出しがコンパイルエラーになる。
* control flow analysisや型ガード関数と組み合わせて型の絞り込みして使うことが多い
