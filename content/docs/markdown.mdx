## 見出し

```markdown
# 見出し1
## 見出し2
### 見出し3
#### 見出し4
##### 見出し5
###### 見出し6
```

<Callout type="warning">
アクセシビリティのために、見出し2から始めてください。
</Callout>

### 見出しのアンカー [#custom-anchor]

`[#<id>]`を付けると、見出しのアンカーをカスタマイズできます。

```markdown
### 見出しのアンカー [#custom-anchor]
```

### **見出し**~の~ _中_ <small>の</small>`Markdown`と目次

見出しの中のMarkdownはそのまま目次にも反映されます。

```markdown
### **見出し**~の~ _中_ <small>の</small>`Markdown`と目次
```

<Callout type="warning">
見出しをリンクにしないでください。`## [見出し](link){:markdown}`のように見出しに直接リンクをつけると、目次をクリックできなくなりUXが悪くなります。
</Callout>

### 目次に出さない見出し [!toc]

`[!toc]`を付けると、目次に出力されません。

```markdown
### 目次に出さない見出し [!toc]
```

```markdown
Hello World, **Bold**, _Italic_, ~~Strike~~
```

Hello World, **Bold**, _Italic_, ~~Strike~~

## コード

`` ` ``で囲った部分はコードとして表示されます。

```markdown
`x`と`y`
```

`` ` ``をコードで表示する場合は、二重の``` `` ```とスペースで囲みます。

```markdown
JavaScriptの`` `string` ``はテンプレートリテラルです。
```

JavaScriptの`` `string` ``はテンプレートリテラルです。

### コード + シンタックスハイライト

コードの最後に`{:<language>}`のように書くと、シンタックスハイライトが適用されます。

```markdown
`print("Hello"){:python}`は`console.log("Hello"){:js}`と同じです。
```

`print("Hello"){:python}`は`console.log("Hello"){:js}`と同じです。

### ページカード

トップレベルのリンクのうち、`./`もしくは`../`で始まり、`.mdx`で終わるものは、ページカードとして表示されます。[^@suin/auto-page-card]

```markdown
[Markdownについて](./markdown.mdx)
```

[Markdownについて](./markdown.mdx)

<Callout type="warning">
`[テキスト](リンク)`のうち、テキストの部分はページカードでは使われませんが、GitHubなどでMarkdownのプレビューを見たときに、リンクとして表示されるようリンクテキストは意味のあるものを指定してください。
</Callout>

## 画像

画像は`/public`ディレクトリ内のパスを指定します。たとえば、`/public/image.png`は`![](/image.png)`のように記述します。

```markdown
![画像の説明](/image.png)
```

## コードブロック

````markdown
```typescript
const message = "Hello World";
```
````

```typescript
const message = "Hello World";
```

### コードブロック + ファイル名

コードブロックのメタ部に`title="<filename>"`を指定すると、ファイル名が表示されます。

````markdown
```typescript title="hello-world.ts"
const message = "Hello World";
```
````

```typescript title="hello-world.ts"
const message = "Hello World";
```

### コードブロック + 行番号

コードブロックのメタ部に`lineNumbers`を指定すると、行番号が表示されます。

````markdown
```typescript lineNumbers
function greet() {
  return "Hello, World!";
}
```
````

```typescript lineNumbers
function greet() {
  return "Hello, World!";
}
```

`lineNumbers=4`のように、行番号を4から始めることもできます。

````markdown
```typescript lineNumbers=4
const message = greet();
console.log(message);
// Prints "Hello, World!"
```
````

```typescript lineNumbers=4
const message = greet();
console.log(message);
// Prints "Hello, World!"
```

### コードブロック + Diff

`[!code --]`や`[!code ++]`を使うことで、コードブロック内の行のdiffのような追加や削除を示すことができます。[^@shikijs/transformers]

{/* We can't use `markdown` for language indication here because it conflicts with the [!code] tags */}

````text
```typescript
console.log("js"); // [!code --]
console.log("ts"); // [!code ++]
```
````

```typescript
console.log("js"); // [!code --]
console.log("ts"); // [!code ++]
```

### コードブロック + 行ハイライト

`[!code highlight]`を使うことで、コードブロック内の特定の行をハイライトできます。[^@shikijs/transformers]

{/* We can't use `markdown` for language indication here because it conflicts with the [!code] tags */}

````text
```typescript
console.log("Not highlighted");
console.log("Highlighted"); // [!code highlight]
console.log("Not highlighted");
```
````

```typescript
console.log("Not highlighted");
console.log("Highlighted"); // [!code highlight]
console.log("Not highlighted");
```

### コードブロック + 単語ハイライト

`[!code word:<word>]`を使うことで、コードブロック内の特定の単語をハイライトできます。[^@shikijs/transformers]<small>(次の例は`[\!code]`となっていますが、`[!code]`に読み替えてください。)</small>

````text
```typescript
// [\!code word:Hello]
const message = "Hello World";
console.log(message); // prints Hello World
```
````

```typescript
// [!code word:Hello]
const message = "Hello World";
console.log(message); // prints Hello World
```

`[!code word:<word>:<lines>]`のように、特定の単語を特定の行数分ハイライトすることもできます。たとえば、`foo:2`はコメントの次の行と更にその次の行の`foo`をハイライトします。<small>(次の例は`[\!code]`となっていますが、`[!code]`に読み替えてください。)</small>

````
```typescript
// [\!code word:foo:2]
// foo bar
// foo bar
// foo bar
// [\!code word:bar:2]
// foo bar
// foo bar
// foo bar
```
````

```typescript
// [!code word:foo:2]
// foo bar
// foo bar
// foo bar
// [!code word:bar:2]
// foo bar
// foo bar
// foo bar
```

## Twoslash

コードブロックのメタ部に`twoslash`を指定すると、TypeScriptのコンパイラーから得られる情報を表示できます。[^twoslash]

### Twoslash 型推論

`^?`で指し示したシンボルの型を表示できます。

````markdown title="Markdown"
```typescript twoslash
const value = [true, 1] as const;
//    ^?
type Key = keyof any;
//   ^?
```
````

```typescript twoslash title="Result"
const value = [true, 1] as const;
//    ^?
type Key = keyof any;
//   ^?
```

### Twoslash コンパイルエラー

`@errors: <error code>`で期待するエラーメッセージを表示できます。

````markdown title="Markdown"
```typescript twoslash
// @errors: 7006
function fn(anyValue) {}
```
````

```typescript twoslash title="Result"
// @noImplicitAny: true
// @errors: 7006
function fn(anyValue) {}
```

### Twoslash コンパイルエラーの抑制

`@noErrors`でコンパイルエラーを無視できます。

````markdown title="Markdown"
```typescript twoslash
// @noErrors
function fn(anyValue) {}
//          ^?
```
````

```typescript twoslash title="Result"
// @noImplicitAny: true
// @noErrors
function fn(anyValue) {}
//          ^?
```

### Twoslash コンパイラーオプションの指定

`@<key>: <value>`でTypeScriptのコンパイラーオプションを指定できます。

````markdown title="Markdown"
```typescript twoslash
// @noImplicitAny: false
function fn(anyValue) {}
//          ^?
```
````

```typescript twoslash title="Result"
// @noImplicitAny: false
function fn(anyValue) {}
//          ^?
```

<Callout type="info">
デフォルトのコンパイラオプションは`tsconfig.twoslash.json`をご覧ください。
</Callout>

### Twoslash 実行結果の表示

`@log`、`@warn`、`@error`で、実行結果を表示できます。

````markdown title="Markdown"
```typescript twoslash
// @noErrors
console.log(123);
// @log: 123
console.warn("メッセージ");
// @warn: メッセージ
const x = value;
// @error: ReferenceError: value is not defined
```
````

```typescript twoslash title="Result"
// @noErrors
console.log(123);
// @log: 123
console.warn("メッセージ");
// @warn: メッセージ
const x = value;
// @error: ReferenceError: value is not defined
```

### Twoslash コード補完

`^|`でコード補完を表示できます。

````markdown title="Markdown"
```typescript twoslash
// @noErrors
"Hello World".ma
//              ^|
```
````

```typescript twoslash title="Result"
// @noErrors
"Hello World".ma
//              ^|
```

### Twoslash トランスパイル結果

`@showEmit`でトランスパイル結果を表示できます。

````markdown title="Markdown"
```typescript twoslash
// @showEmit
export type Type = string | number;
export function fn<T extends Type>(value: T): T {
  return value;
}
```
````

```typescript twoslash title="Result"
// @showEmit
export type Type = string | number;
export function fn<T extends Type>(value: T): T {
  return value;
}
```

### Twoslash 型定義ファイルの表示

`@showEmit`、`@declaration: true`、`@showEmittedFile: index.d.ts`をセットで指定すると、型定義ファイルへのトランスパイル結果が表示できます。

````markdown title="Markdown"
```typescript twoslash
// @showEmit
// @declaration: true
// @showEmittedFile: index.d.ts
export type Type = string | number;
export function fn<T extends Type>(value: T): T {
  return value;
}
```
````

```typescript twoslash title="Result"
// @showEmit
// @declaration: true
// @showEmittedFile: index.d.ts
export type Type = string | number;
export function fn<T extends Type>(value: T): T {
  return value;
}
```

### Twoslash 単語ハイライト

2つ以上連続する`^`で単語をハイライトできます。

````markdown title="Markdown"
```typescript twoslash
const message = "Hello World";
//               ^^^^^
console.log(message);
```
````

```typescript twoslash title="Result"
const message = "Hello World";
//               ^^^^^
console.log(message);
```

<Callout type="info">
`^`をコメントとして使いたい場合は、`^`に続けてスペース以外の文字を書いてください。`^^^ text`のように間にスペースがあるとハイライトになるので注意してください。

````markdown title="Markdown"
```typescript twoslash
1234567890;
//     ^^^これはコメントです
1234567890;
//     ^^^ これはハイライトです
```
````

```typescript twoslash title="Result"
1234567890;
//     ^^^これはコメントです
1234567890;
//     ^^^ これはハイライトです
```
</Callout>

### Twoslash 割愛

`---cut---`を使うと、それ以前のコードを非表示にできます。

````markdown title="Markdown"
```typescript twoslash
"line 1";
// ---cut---
"line 3";
```
````

```typescript twoslash title="Result"
"line 1";
// ---cut---
"line 3";
```

`---cut-after---`を使うと、それ以降のコードを非表示にできます。

````markdown title="Markdown"
```typescript twoslash
"line 1";
// ---cut-after---
"line 3";
```
````

```typescript twoslash title="Result"
"line 1";
// ---cut-after---
"line 3";
```

## コメント

コメントは`{/* コメント */}{:mdx}`のように書きます。これはMDXのコメントで、HTMLのコメント`<!-- コメント -->{:html}`は使用できません。

```mdx
{/* コメント */}
```
